# PromptGen

PromptGen — это веб-приложение и PWA для удобной сборки промтов для ИИ.

Приложение позволяет:
- использовать готовые заготовки под типовые задачи;
- собирать собственные промты в свободном режиме;
- сохранять шаблоны локально;
- работать как установленное PWA-приложение;
- использовать приложение без аккаунта и без сервера.

Все данные хранятся локально на устройстве пользователя.

---

## Возможности

### Основные сценарии
- **Заготовки** — готовые сценарии с предзаполненными полями.
- **Свободный режим** — ручная сборка промта из блоков.
- **Редактор** — доработка готовой заготовки под себя.
- **Библиотека** — сохранённые шаблоны, избранное, история.
- **Настройки** — тема, размер текста, формат ответа по умолчанию, экспорт и импорт.
- **PWA** — установка на устройство, standalone-режим, офлайн-доступ к интерфейсу и локальным данным.

### Что умеет приложение
- предпросмотр итогового промта в реальном времени;
- копирование итогового текста;
- сохранение шаблонов в локальное хранилище;
- импорт и экспорт шаблонов в JSON;
- работа без авторизации;
- адаптивный интерфейс для десктопа и мобильных устройств;
- светлая и тёмная темы;
- поддержка установки как PWA.

## Архитектура хранения данных

Приложение не использует backend и не отправляет пользовательские данные на сервер.

### Локальное хранение
- **IndexedDB**
  - пользовательские шаблоны
  - история
- **localStorage**
  - настройки интерфейса
  - избранные заготовки
  - недавние элементы
  - служебные PWA-флаги и dismiss-состояния

---

## PWA

PromptGen реализован как полноценное PWA-приложение.

### Что уже поддерживается
- manifest с корректными `id`, `scope`, `start_url`
- установка на устройство
- standalone-режим
- иконки приложения
- maskable-иконка
- shortcuts в manifest
- screenshots в manifest
- offline shell через service worker
- контролируемое обновление приложения
- защита от потери несохранённого ввода при обновлении
- поддержка GitHub Pages

### Особенности
- сервис-воркер регистрируется через `vite-plugin-pwa`
- стратегия: `generateSW`
- обновления — в режиме **prompt**, без внезапного автообновления во время работы
- для GitHub Pages предусмотрен `404.html` fallback для SPA-маршрутов
